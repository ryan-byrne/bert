    Getting Started #

    This page is meant to get you acquinted with Bert

    Available Features #

    Bert comes with a wide array of features such as:

    • Booking Time in the Beta Lab
    • Completing Trainings
    • Checking Tool Availability
    • …and much more!

    For a full list, select the Features tab in the menu.

    Logging In #

    1. When you first open Bert, you will be redirected to a Google Oauth2 Login Page

    2. Select your Berkeley Carroll (@berkeleycarroll.org) Account

    IMPORTANT: Do not select another email (such as your personal). This will block you from being to access Bert.

    1. Bert will now ask for permission to access various elements of your Google Account (such as Calendar, Classroom, etc.). Select allow.

    Navigating Bert #

    Bert can be explored using the Navigation Bar at the top of the screen.

    At any time, you can select the Bert Logo at the top left of the screen to be returned to the Home Page.

    Logging Out #

    If you wish to sign out of Bert:

    1. Select the dropdown menu with your name and press Sign Out.

    1. On the screen that appears, confirm you wish to sign out by pressing Yes, Sign Out
